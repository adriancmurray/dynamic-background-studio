# Dynamic Background Studio

An extendable, high-performance library of interactive website backgrounds built in **Svelte 5** and styled with **Vanilla CSS**, featuring a native-feeling macOS/iOS-inspired customization playground.

---

## Key Features

- **Portability**: All background components are written in pure Svelte 5 and styled with scoped Vanilla CSS. You can copy the code of any component directly into your project with **zero dependencies**.
- **Interactive Playground**: A responsive, wowed desktop/mobile sizer with translucent panel layers, theme toggles, and live parameter adjusters.
- **Extendable Schema**: Adding a new background is as simple as creating a new `.svelte` file and registering its controls in `presetCatalog.ts`.
- **Performance Optimized**: Uses continuous 2D Canvas buffers, minimal garbage collection allocation inside frame loops, hardware-accelerated CSS animations, and built-in hooks for `prefers-reduced-motion` compliance.

---

## The Background Collection

1. **Interactive Dot Field (`DotField.svelte`)**: A depth-mapped grid of dots reacting dynamically to cursor movement (repulsion or attraction) and custom pulse wave ripples.
2. **Dynamic Constellation (`Constellation.svelte`)**: Floating physics-based nodes that draw thin link lines to neighboring nodes when they float within range.
3. **Perlin Flow Field (`FlowField.svelte`)**: Floating stream particles tracing vectors generated by a lightweight 3-octave Fractional Brownian Motion (fbm) noise field.
4. **CSS Aurora Glass (`Aurora.svelte`)**: Multiple highly blurred moving gradient blobs utilizing CSS variable bindings and hardware-accelerated animations.

## Local Development

To run the builder application locally:

```bash
# Clone the repository
git clone https://github.com/your-username/dynamic-background-studio.git
cd dynamic-background-studio

# Install dependencies
npm install

# Start the Vite development server
npm run dev

# Run Svelte and TypeScript checks
npm run check

# Build production distribution bundle
npm run build
```
